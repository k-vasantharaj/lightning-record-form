🧾✨ Account Editor - Lightning Web Component (LWC) 🚀
📖 Description

A Salesforce Lightning Web Component that allows users to view and update Account records using the standard lightning-record-form ⚡

This project demonstrates real-world usage of:

🔹 Standard Salesforce UI components
🔹 Layout-based form rendering (Compact 🧩 & Full 📄)
🔹 Toast notifications for success ✅ and error ❌ handling

💡 Users can quickly edit Account details in a clean and structured UI.

🌟 Key Features

✨ Uses lightning-record-form for automatic CRUD handling
📐 Supports Compact layout (2 columns) for quick edits
📄 Supports Full layout for detailed record view
🔔 Displays toast messages for success & error feedback
🎨 Styled with Salesforce Lightning Design System (SLDS)

⚙️ Installation

📥 Clone this repository

🛠 Deploy the component using:

SFDX
VS Code

🧩 Open Lightning App Builder

📌 Drag & drop the component onto:

Record Page (Recommended ⭐)
App Page
Home Page
▶️ Usage

📍 Open any Account record page

🧩 Add the Account Editor component

✏️ Edit the fields and click Save

📢 Output:

✅ Account updated successfully
❌ Error message if something goes wrong
🛠 Configuration

You are using layout-based rendering instead of hardcoded fields 👇

layout-type="Compact"
columns="2"

⚙️ Options you can tweak:

🔹 layout-type="Compact" → Faster editing ⚡
🔹 layout-type="Full" → Detailed layout 📄
🔹 columns="2" → Better UI structure 📐
